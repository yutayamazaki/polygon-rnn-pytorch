# Polygon-RNN PyTorch

An implementation of PolygonRNN using PyTorch.

## [Annotating Object Instances with a Polygon-RNN](https://arxiv.org/abs/1704.05548)

### Hyper parameters

|  |  |
|--|--|
| bacth size | 8    |
| lr         | 1e-4 (*0.1 in each 10 epochs) |
| optimizer  |  Adam \beta_1=0.9, \beta_2=0.999 |
